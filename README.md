# cravegaurd
CraveGuard is a mindful ordering app that breaks the cycle of impulsive eating. It uses behavioral nudges to intercept unhealthy choices, suggesting nutritious "smart swaps" instead. With a food scanner and a "last-chance" prompt at checkout, it ensures every meal is a conscious decision. The mindful pause your health deserves.
